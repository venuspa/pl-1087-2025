# Apresentação PL 1087/2025

Este repositório contém a apresentação HTML/CSS/JS do PL 1087/2025, convertida e ajustada a partir do PPTX fornecido.

## Acesso Público (GitHub Pages)

A apresentação está publicada via GitHub Pages e pode ser acessada diretamente pelo link:

- https://venuspa.github.io/pl-1087-2025/

Caso o conteúdo não apareça imediatamente, aguarde alguns minutos e recarregue a página (o build do Pages pode levar um tempo para propagar).

## Conteúdo do Projeto

- `index.html`: arquivo principal da apresentação
- `styles.css`: estilos com layout, cores e responsividade
- `script.js`: lógica de navegação e atalhos
- `assets/`: imagens e mídias
- `pptx_extract/`: textos e metadados extraídos do PPTX
- `contexto.txt`: observações e anotações de contexto

## Execução Local

- Requisitos: Node.js (para servir via `serve`) ou qualquer servidor estático
- Servidor local: `npx serve -p 5173` e abra `http://localhost:5173/`

## Navegação Rápida

- Setas (`←`/`→`): muda de slide
- `Home`: volta ao início
- `End`: vai ao último slide

## Impressão/Exportação

- No navegador, use `Ctrl+P` e selecione "Salvar como PDF"
- Ative fundos e gráficos para manter o visual

## Publicação via Git/CLI

- Repositório GitHub: https://github.com/venuspa/pl-1087-2025
- Branch padrão: `main`
- GitHub Pages: habilitado a partir da `main` (raiz `/`)
